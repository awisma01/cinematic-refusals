About the Map

This interactive map visualizes the network of Israeli checkpoints and their representation in Palestinian cinema. Each marker corresponds to a real checkpoint, with details about its location, status, and how it appears in films.

Click any marker to view details and analysis. Use the Archive panel to browse and filter all checkpoints and films. You can also search by name or location using the search bar above the map.

Space is not neutral. By linking geography with film, this project shows how Palestinian filmmakers document, resist, and reimagine the spatial realities of occupation.
