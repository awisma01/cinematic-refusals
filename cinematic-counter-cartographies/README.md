# Cinematic Counter-Cartographies

## Overview
Cinematic Counter-Cartographies is a digital humanities project that examines the concept of formal refusal in Palestinian cinema from 2000 to 2025. The project aims to explore the intersections of cinema, geography, and politics, providing a platform for understanding the narratives and representations within Palestinian films.

## Project Structure
The project consists of the following key files and directories:

- **index.html**: The main entry point of the website, containing the homepage content and navigation.
- **about.html**: Provides information about the project, including context, methodology, and theoretical frameworks.
- **map.html**: Features an interactive map, film archive, timeline slider, and statistics related to checkpoints and films.
- **checkpoints.html**: Contains information or analysis related to checkpoints in Palestinian cinema.
- **analysis.html**: Offers detailed analyses of specific films relevant to the project.
- **not-mapped.html**: Discusses films or topics that are not included in the mapping.

### Directories
- **css/**: Contains the main stylesheet (`style.css`) for the project, defining the overall look and feel of the website.
- **js/**: Contains the main JavaScript file (`main.js`), which includes scripts for general functionality across the site.
- **data/**: 
  - `films.json`: Contains data about films relevant to the project, including titles, directors, and other metadata.
  - `locations.geojson`: Contains geographical data in GeoJSON format, used for mapping checkpoints and film locations.
- **docs/**: 
  - `methodology.md`: Outlines the methodology used in the project, providing insights into the research approach.

## Installation
To set up the project locally, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd cinematic-counter-cartographies
   ```
3. Open `index.html` in your web browser to view the project.

## Usage
Explore the various sections of the project through the navigation menu. Each section provides unique insights into Palestinian cinema and its geographical context.
